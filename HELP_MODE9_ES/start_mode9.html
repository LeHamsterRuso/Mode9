START_MODE9()

START_MODE9( <id de la cámara> , <OFFSET m9.z> , <región> )

Descripción:

Activa el render del escenario 3D del modo 9. Es el equivalente a start_mode8(), con la misma firma de tres parámetros. A partir de su llamada, la capa de FONDO de DIV2 deja de ser el gráfico 2D y pasa a ser el escenario tridimensional que la DLL dibuja cada frame.

Antes de llamarla hay que cargar el escenario con LOAD_WLD9(). La cámara se sitúa en la posición del proceso indicado (sus campos x, y en el plano del suelo y angle como giro horizontal), a la altura y con la inclinación definidas en la estructura m9.

Parámetros:

ParámetroSignificado
id de la cámaraCódigo identificador del proceso en cuya posición se sitúa la cámara. Normalmente id (el propio proceso que controla el juego).
OFFSET m9.zDirección de la estructura m9 (se pasa &m9.z, el primer campo). La DLL lee de ahí m9.height (altura del ojo) y m9.angle (pitch) cada frame. Equivale al BIND de la estructura.
regiónRegión de pantalla: 0 = pantalla completa; 1..15 = región definida previamente con MODE9_REGION() (varias llamadas crean una pantalla partida por regiones, como las ventanas del modo 8).

Retorna: 1 (siempre que se active).

Nota: Una DLL no puede crear STRUCTs ni c_types del lenguaje DIV2, por eso la estructura m9 (campos z, camera, height, angle) y la constante c_m9 se declaran en el PRG y se pasan a la DLL con esta función y con MODE9_SET().
Programa de ejemplo: 
PROGRAM mode9_minimo;
IMPORT "MODE9.DLL";
CONST
    c_m9 = 200;
GLOBAL
    STRUCT m9
        z; camera; height; angle;
    END
BEGIN
    set_mode(320200);
    set_fps(60, 0);
    load_fpg("FPG/MODE9.FPG");
    // Cargar el escenario: malla + textura + escala
    LOAD_WLD9("CASTLE.OBJ", "MAP/CASTLE.MAP", 100);
    // Situar la cámara a 70 uds de altura
    MODE9_SET(70, 0);
    // Activar el render en pantalla completa
    START_MODE9(id, OFFSET m9.z, 0);
    LOOP
        if (key(_up))   advance(20);  end
        if (key(_down)) advance(-20); end
        if (key(_left))  angle += 4000; end
        if (key(_right)) angle -= 4000; end
        FRAME;
    END
END

Este programa carga un escenario 3D y deja moverse por él. Fíjate en que mover la cámara es tan sencillo como en un juego 2D normal: advance() la adelanta y cambiar el angle del proceso la hace girar, exactamente igual que harías con cualquier proceso, solo que ahora lo que ves es el mundo 3D desde su posición.
