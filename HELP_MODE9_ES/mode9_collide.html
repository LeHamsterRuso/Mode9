MODE9_COLLIDE()

MODE9_COLLIDE( <id del proceso> )

Descripción:

Empuja CUALQUIER proceso fuera de las paredes (deslizando a lo largo de ellas) y reescribe sus campos x, y. Pensada para que los NPCs y enemigos respeten las paredes igual que la cámara.

Usa el campo radius del proceso (30 por defecto). Si MODE9_STEP() es mayor que 0, ademas apoya al proceso sobre el suelo.

Parámetros:

ParámetroSignificado
id del procesoIdentificador del proceso (NPC, enemigo...) que se quiere mantener fuera de las paredes.

Retorna: 1 si lo desplazó, 0 si no.

Programa de ejemplo: 
PROCESS enemigo()
BEGIN
    radius = 40;
    LOOP
        advance(10);            // moverse
        MODE9_COLLIDE(id);      // ...sin atravesar paredes
        // colocarlo como billboard en el escenario
        MODE9_BILLBOARD(id, x, y, z, 100);
        FRAME;
    END
END
