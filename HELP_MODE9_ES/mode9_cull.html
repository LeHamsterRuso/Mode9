MODE9_CULL()

MODE9_CULL( <modo> )

Descripción:

Cada cara (polígono) de un modelo tiene un "derecho" y un "revés". Para ir más rápido, el modo 9 no dibuja por defecto las caras que están de espaldas a la cámara (las que no podrías ver de todas formas, como la cara trasera de una pared). A eso se le llama descartar caras traseras. Esta función te deja cambiar ese comportamiento.

Lo normal es dejarlo como está. Solo necesitarás tocarlo si al entrar en un sitio las paredes "desaparecen" o se ven transparentes: eso pasa cuando el modelo tiene las caras del revés, y se arregla con el modo 0 o el 2.

Parámetros:

ParámetroSignificado
modo0 = dibujar todas las caras (las veas o no, más lento pero nunca desaparece nada); 1 = no dibujar las traseras (por defecto, lo más rápido); 2 = al revés que el 1, útil para escenarios modelados "del derecho hacia dentro", como el interior de una habitación.

Retorna: 0.

Nota: Truco: si entras en una sala y no ves las paredes desde dentro, prueba MODE9_CULL(0) (siempre funciona) o MODE9_CULL(2).
Programa de ejemplo: 
MODE9_CULL(1);   // descartar caras traseras (por defecto, más rápido)
Ver: LOAD_WLD9()
