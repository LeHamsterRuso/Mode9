MODE9_STEP()

MODE9_STEP( <altura> )

Descripción:

Define la altura máxima (uds DIV2) que el jugador sube por sí solo: escaleras y rampas. Los muros más bajos que ese valor no bloquean (se suben como un peldaño) y la cámara se apoya en el suelo que tiene debajo. Con 0 se desactiva.

La DLL solo SUBE la z al pisar suelo; la GRAVEDAD (la caída) la gestiona el PRG restando a la z cada frame.

Parámetros:

ParámetroSignificado
alturaAltura máxima de peldaño/escalón que se sube automáticamente, en uds DIV2. 0 = desactivado.

Retorna: 0.

Nota: Requiere las colisiones activadas: no funciona si has hecho MODE9_COLLISION(0).
Programa de ejemplo: 
MODE9_STEP(70);   // sube escalones de hasta 70 uds

// La gravedad la aplica el PRG cada frame:
fall_v += 4;            // acelerar la caída
m9.height -= fall_v;    // bajar
if (m9.height < 70)     // tope al tocar suelo
    m9.height = 70; fall_v = 0;
end
