LOAD_OBJ9()

LOAD_OBJ9( <fichero OBJ> , <textura MAP> , <escala> )

Descripción:

Carga un modelo 3D de objeto (un NPC o un prop tridimensional, no el escenario). A diferencia del escenario, puedes tener varios modelos cargados a la vez y dibujar cada uno en muchas posiciones con DRAW_OBJ9().

Devuelve el handle al INSTANTE; la carga del fichero se hace EN SEGUNDO PLANO (no bloquea, un fichero por frame). El modelo no se dibuja hasta que termina de cargar: DRAW_OBJ9 lo ignora mientras tanto, así que el objeto aparece solo cuando esta listo. Usa MODE9_OBJ9_READY() si necesitas saber cuando esta. Los modelos se conservan hasta STOP_MODE9().

Parámetros:

ParámetroSignificado
fichero OBJFichero .OBJ del modelo (Y hacia arriba, triangulado; UVs opcionales).
textura MAPTextura .MAP del modelo. "" = flat-shading con la rampa de sombreado del mundo.
escalaUnidades DIV2 por unidad OBJ del modelo (igual concepto que el unitscale del escenario).

Retorna: Un handle 1..N del modelo (0 si no hay slots libres).

Nota: La carga en segundo plano evita que se dispare el watchdog (excepción 142) aunque cargues muchos modelos.
Programa de ejemplo: 
GLOBAL
    h_npc;
BEGIN
    ...
    h_npc = LOAD_OBJ9("KNIGHT.OBJ", "MAP/KNIGHT.MAP", 100);
    LOOP
        // dibujar el modelo en el mundo cada frame
        DRAW_OBJ9(h_npc, 0, 500, 0, 0);
        FRAME;
    END
END
