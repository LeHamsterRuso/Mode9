SET_ENV_COLOR9()

SET_ENV_COLOR9( <r> , <g> , <b> )

Descripción:

Elige el color del fondo, es decir, lo que se ve por detrás del escenario donde no hay ni paredes ni suelo (el "cielo" liso, si no has puesto un cielo con dibujo). Ese mismo color es ademas el de la niebla, así que con el decides el ambiente general de la escena: un azul claro para un dia despejado, un gris para un dia nublado, un tono oscuro para un interior o una noche.

Cada color se da con sus tres componentes rojo, verde y azul, de 0 a 100: 0 es nada de ese color y 100 el máximo (así, 0,0,0 es negro y 100,100,100 es blanco).

Si tienes puesto un cielo con dibujo (LOAD_SKY9()), ese cielo tapa el color de fondo, pero el color sigue usandose para la niebla.

Parámetros:

ParámetroSignificado
rComponente rojo del color de fondo/niebla, 0..100.
gComponente verde, 0..100.
bComponente azul, 0..100.

Retorna: 0.

Programa de ejemplo: 
SET_ENV_COLOR9(30, 35, 50);   // cielo crepuscular azulado
Ver: SET_FOG9() - LOAD_SKY9() - SKY9_OFF()
