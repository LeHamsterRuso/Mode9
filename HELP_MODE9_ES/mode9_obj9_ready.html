MODE9_OBJ9_READY()

MODE9_OBJ9_READY( <handle> )

Descripción:

Consulta el estado de carga de un modelo de LOAD_OBJ9() o LOAD_OBJ9_ANIM(), que cargan en segundo plano.

Útil para mostrar una barra de "Cargando..." o esperar a que todo este listo antes de empezar la partida. No es obligatorio: DRAW_OBJ9 ya ignora por su cuenta los modelos que no están listos.

Parámetros:

ParámetroSignificado
handleHandle del modelo a consultar.

Retorna: 1 = cargado; 0 = aun cargando; -1 = handle invalido o fallo.

Programa de ejemplo: 
h_npc = LOAD_OBJ9("KNIGHT.OBJ", "MAP/KNIGHT.MAP", 100);
// Esperar a que el modelo este listo mostrando un mensaje
while (MODE9_OBJ9_READY(h_npc) == 0)
    write(0, 160, 100, 4, "Cargando...");
    FRAME;
end
delete_text(all_text);
