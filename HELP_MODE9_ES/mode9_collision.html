MODE9_COLLISION()

MODE9_COLLISION( <on> )

Descripción:

Activa o desactiva las colisiones de la cámara contra las paredes. Con las colisiones activadas, advance() y xadvance() no atraviesan los muros: la distancia mínima a la pared es el campo radius del proceso cámara.

Las colisiones están activadas por defecto, así que solo necesitas llamar a esta función con 0 si quieres DESACTIVARLAS (por ejemplo, un modo "noclip").

Parámetros:

ParámetroSignificado
on1 = colisiones activas (no atraviesa paredes); 0 = sin colisiones (atraviesa paredes).

Retorna: 0.

Nota: Las escaleras y rampas de MODE9_STEP() solo funcionan con las colisiones activadas.
Programa de ejemplo: 
MODE9_COLLISION(0);   // modo noclip: atravesar paredes
MODE9_COLLISION(1);   // restaurar colisiones (estado por defecto)
