MODE9_FARCLIP()

MODE9_FARCLIP( <distance> )

Description:

Defines the maximum draw distance: geometry beyond distance DIV2 units is not drawn (pop-in effect). Affects the scene, OBJ models and billboards. 0 means no limit (default).

If you enable fog with SET_FOG9(), the far clip is activated automatically using the fog's far value (what lies beyond the fog is the background colour and is not drawn).

Parameters:

ParameterMeaning
distanceMaximum draw distance in DIV2 units. 0 = no limit.

Returns: 0.

Example program: 
MODE9_FARCLIP(1000);   // do not draw anything beyond 1000 units
See: SET_FOG9() - MODE9_FOG_OFF()
