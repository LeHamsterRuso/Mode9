MODE9_SETTEX()

MODE9_SETTEX( <MAP texture> )

Description:

Replaces the texture of material 0 (the default material loaded by LOAD_WLD9()) on its own. It is equivalent to MODE9_MATERIAL(0, map).

Useful for re-texturing the scene without reloading the mesh (for example, changing a level's appearance based on the time of day).

Parameters:

ParameterMeaning
MAP texture.MAP file to replace the material 0 texture.

Returns: 1 if the texture loads successfully, 0 if it fails.

Example program: 
// Change the scene texture without reloading the OBJ
MODE9_SETTEX("MAP/CASTLE2.MAP");
See: MODE9_MATERIAL() - LOAD_WLD9()
