MODE9_LOADFPG()

MODE9_LOADFPG( <FPG file> )

Description:

Loads an FPG INSIDE the DLL so that the DLL itself draws billboards (processes with ctype c_m9) in EACH viewport. Required for sprites to display correctly in split-screen.

Without this call, billboards are drawn through DIV2's c_screen path, which only covers one view. Normally you load the same FPG already used by the game with load_fpg().

Parameters:

ParameterMeaning
FPG file.FPG file containing the billboard graphics. Must include the graphic codes that c_m9 processes will use.

Returns: The number of graphics in the loaded FPG.

Example program: 
load_fpg("FPG/MODE9.FPG");      // game FPG (for DIV2)
MODE9_LOADFPG("FPG/MODE9.FPG"); // same FPG inside the DLL (for split-screen)
