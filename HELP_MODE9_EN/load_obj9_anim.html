LOAD_OBJ9_ANIM()

LOAD_OBJ9_ANIM( <base> , <MAP texture> , <scale> , <nframes> )

Description:

Loads an animated 3D model, just as LOAD_OBJ9() loads a still one. The animation is stored as several separate poses (for example, the different postures of a meowing cat), and each pose is its own .OBJ file.

The pose files are all named the same except for a number at the end, starting at 0. You pass the shared name (the base) and how many poses there are (nframes), and mode 9 loads them all. For example, with base = "CAT" and nframes = 10, it looks for the files CAT0.OBJ, CAT1.OBJ ... up to CAT9.OBJ.

To look smooth, the animation does not need one pose per instant: a few key poses are enough, because mode 9 fills in the steps between them by gradually moving each point from one pose to the next (this is called morphing or interpolation). Ten poses are enough for a smooth walk cycle.

Important: every pose must be the same model, only with its points placed differently; that is, they must all have the same number of vertices, the same triangles and the same UVs. Only where each point sits changes, not how many there are. (In Blender: model one pose and move its vertices for the others, without adding or removing anything.)

The whole animation takes a single model slot, no matter how many poses it has. The files load in the background, one per frame (it does not block the game), and you draw it with DRAW_OBJ9_ANIM().

Parameters:

ParameterMeaning
baseShared name of the poses, without the number or extension (e.g. "CAT" → CAT0.OBJ, CAT1.OBJ...). Remember the DOS limit: base + the number must total 8 characters or fewer (CAT + 0..9 fits easily).
MAP textureThe model's .MAP texture (or "" for flat shading with no texture).
scaleDIV2 units per OBJ unit (same as in LOAD_OBJ9).
nframesHow many poses the animation has (how many files, numbered 0..nframes-1).

Returns: A handle 1..N for the animated model (0 if no free slots).

Example program: 
GLOBAL
    h_cat;
BEGIN
    // 10 poses: CAT0.OBJ .. CAT9.OBJ (same model, vertices moved)
    h_cat = LOAD_OBJ9_ANIM("CAT", "MAP/CAT.MAP", 100, 10);
    ...
END
