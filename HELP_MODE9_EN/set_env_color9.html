SET_ENV_COLOR9()

SET_ENV_COLOR9( <r> , <g> , <b> )

Description:

Sets the background colour — what you see behind the scene where there are no walls or floors (the plain "sky", when no sky texture has been loaded). This same colour is also used for fog, so with it you define the general atmosphere of the scene: a light blue for a clear day, a grey for an overcast day, a dark tone for an interior or nighttime.

Each colour is given as its three red, green and blue components, from 0 to 100: 0 is none of that colour and 100 is the maximum (so 0,0,0 is black and 100,100,100 is white).

If you have set a sky texture (LOAD_SKY9()), that sky covers the background colour, but the colour is still used for fog.

Parameters:

ParameterMeaning
rRed component of the background/fog colour, 0..100.
gGreen component, 0..100.
bBlue component, 0..100.

Returns: 0.

Example program: 
SET_ENV_COLOR9(30, 35, 50);   // bluish twilight sky
See: SET_FOG9() - LOAD_SKY9() - SKY9_OFF()
