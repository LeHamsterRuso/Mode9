DRAW_OBJ9()

DRAW_OBJ9( <handle> , <x> , <y> , <z> , <angle> )

Description:

Queues an instance of model handle (returned by LOAD_OBJ9()) at world position (x, y, z) in DIV2 units, with the given orientation. Must be called EVERY FRAME, before FRAME, for each instance you want to see.

The queue is cleared automatically after all views have been rendered, so the same model can be drawn many times at different positions in the same frame.

Parameters:

ParameterMeaning
handleModel handle returned by LOAD_OBJ9.
x, y, zWorld position in DIV2 units (x, y on the ground; z = height).
angleOrientation in thousandths of a degree (DIV2 convention; 180000 = 180 degrees).

Returns: 0.

Note: While the model is still loading in the background, DRAW_OBJ9 silently ignores it: the object appears automatically when loading completes.
Example program: 
// Draw three knights in a row, facing south
DRAW_OBJ9(h_npc, -200, 500, 0, 0);
DRAW_OBJ9(h_npc,    0, 500, 0, 0);
DRAW_OBJ9(h_npc,  200, 500, 0, 0);
See: LOAD_OBJ9() - DRAW_OBJ9_ANIM()
