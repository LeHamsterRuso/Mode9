MODE9_COLLISION()

MODE9_COLLISION( <on> )

Description:

Enables or disables camera collision against walls. With collisions enabled, advance() and xadvance() do not pass through walls: the minimum distance to a wall is the camera process's radius field.

Collisions are enabled by default, so you only need to call this function with 0 if you want to DISABLE them (for example, a "noclip" mode).

Parameters:

ParameterMeaning
on1 = collisions active (does not pass through walls); 0 = no collisions (passes through walls).

Returns: 0.

Note: Stairs and ramps from MODE9_STEP() only work when collisions are enabled.
Example program: 
MODE9_COLLISION(0);   // noclip mode: walk through walls
MODE9_COLLISION(1);   // restore collisions (default state)
See: MODE9_COLLIDE() - MODE9_STEP()
