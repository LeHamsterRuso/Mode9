MODE9_COLLIDE()

MODE9_COLLIDE( <process id> )

Description:

Pushes ANY process out of the walls (sliding along them) and rewrites its x, y fields. Designed to make NPCs and enemies respect walls just like the camera does.

Uses the process's radius field (30 by default). If MODE9_STEP() is greater than 0, it also rests the process on the floor beneath it.

Parameters:

ParameterMeaning
process idIdentifier of the process (NPC, enemy...) to keep outside walls.

Returns: 1 if it was displaced, 0 if not.

Example program: 
PROCESS enemy()
BEGIN
    radius = 40;
    LOOP
        advance(10);            // move
        MODE9_COLLIDE(id);      // ...without passing through walls
        // place as a billboard in the scene
        MODE9_BILLBOARD(id, x, y, z, 100);
        FRAME;
    END
END
