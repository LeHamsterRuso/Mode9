LOAD_OBJ9()

LOAD_OBJ9( <OBJ file> , <MAP texture> , <scale> )

Description:

Loads a 3D object model (an NPC or a three-dimensional prop, not the scene). Unlike the scene, you can have multiple models loaded at the same time and draw each of them at many positions with DRAW_OBJ9().

Returns the handle IMMEDIATELY; the file is loaded IN THE BACKGROUND (non-blocking, one file per frame). The model is not drawn until loading finishes: DRAW_OBJ9 ignores it until then, so the object appears only when it is ready. Use MODE9_OBJ9_READY() if you need to know when it is. Models are kept until STOP_MODE9().

Parameters:

ParameterMeaning
OBJ file.OBJ file for the model (Y up, triangulated; UVs optional).
MAP textureThe model's .MAP texture. "" = flat-shading with the world shading ramp.
scaleDIV2 units per OBJ unit for this model (same concept as the scene unitscale).

Returns: A handle 1..N for the model (0 if no free slots).

Note: Background loading avoids the watchdog exception (142) even when loading many models.
Example program: 
GLOBAL
    h_npc;
BEGIN
    ...
    h_npc = LOAD_OBJ9("KNIGHT.OBJ", "MAP/KNIGHT.MAP", 100);
    LOOP
        // draw the model in the world each frame
        DRAW_OBJ9(h_npc, 0, 500, 0, 0);
        FRAME;
    END
END
