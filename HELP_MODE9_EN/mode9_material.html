MODE9_MATERIAL()

MODE9_MATERIAL( <index> , <MAP texture> )

Description:

Assigns a .MAP texture to a scene material. An OBJ can declare several materials with usemtl; they are numbered in order of appearance: 0, 1, 2... Material 0 is the one loaded by LOAD_WLD9().

Call it after LOAD_WLD9, once for each material you want to customise. If a material receives no texture of its own, it falls back to material 0.

Parameters:

ParameterMeaning
indexMaterial number (0..15) in the order of the usemtl entries in the OBJ.
MAP texture.MAP file to assign. If that material already had a texture, it is released first (no memory leak).

Returns: 1 if the texture loads successfully, 0 if it fails.

Example program: 
// CASTLE.OBJ has 3 materials: 0=Castle, 1=Stairs, 2=Water
LOAD_WLD9("CASTLE.OBJ", "MAP/CASTLE.MAP", 100);  // material 0
MODE9_MATERIAL(1, "MAP/STAIRS.MAP");             // staircase steps
MODE9_MATERIAL(2, "MAP/WATER.MAP");              // water surrounding the island

If a .MAP file does not exist yet, that material falls back to material 0 without causing a failure.

See: LOAD_WLD9() - MODE9_SETTEX()
