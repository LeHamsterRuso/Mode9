MODE9_SCRH()

MODE9_SCRH()

Description:

Returns the height of the active video resolution in pixels. It complements MODE9_SCRW() for adapting the UI to the actual resolution.

Returns: The height in pixels of the current screen.

Example program: 
scr_y = MODE9_SCRH();
// Status bar at the bottom of the screen, regardless of resolution
write(0, 4, scr_y - 10, 0, "HEALTH");
See: MODE9_SCRW() - MODE9_SPLIT()
