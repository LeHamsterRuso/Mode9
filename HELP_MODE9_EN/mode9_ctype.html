MODE9_CTYPE()

MODE9_CTYPE( <n> )

Description:

Sets the value of the c_m9 constant, the "logical" ctype that marks a process as an automatic billboard (analogous to c_m7 in mode 7). It must MATCH the c_m9 constant declared in the PRG. The default value is 200.

You only need to call it if for some reason you want to use a value other than 200 in your PRG; in that case, change the CONST and inform the DLL with this function.

Parameters:

ParameterMeaning
nNew value of c_m9 (must be > 2 to avoid clashing with c_screen/c_scroll/c_m7).

Returns: 0.

Example program: 
CONST c_m9 = 200;       // in the PRG
...
MODE9_CTYPE(c_m9);      // inform the DLL (only if you change the default value)
See: MODE9_BILLBOARD() - MODE9_LOADFPG()
