MODE9_CULL()

MODE9_CULL( <mode> )

Description:

Every face (polygon) of a model has a "front" and a "back". For speed, mode 9 does not draw faces that are turned away from the camera by default (faces you would not see anyway, like the back side of a wall). This is called back-face culling. This function lets you change that behaviour.

The default is fine for most scenes. You only need to touch it if walls "disappear" or appear transparent when you enter an area: that happens when the model's faces are flipped, and it is fixed with mode 0 or mode 2.

Parameters:

ParameterMeaning
mode0 = draw all faces (whether visible or not — slower but nothing ever disappears); 1 = skip back faces (default, fastest); 2 = opposite of 1, useful for scenes modelled "inside-out", such as the interior of a room.

Returns: 0.

Tip: If you enter a room and cannot see the walls from inside, try MODE9_CULL(0) (always works) or MODE9_CULL(2).
Example program: 
MODE9_CULL(1);   // skip back faces (default, fastest)
See: LOAD_WLD9()
