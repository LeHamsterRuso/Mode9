MODE9_STEP()

MODE9_STEP( <height> )

Description:

Defines the maximum height (DIV2 units) the player climbs automatically: stairs and ramps. Walls shorter than this value do not block movement (they are stepped over like a stair) and the camera rests on the floor beneath it. Setting 0 disables it.

The DLL only RAISES z when standing on a floor; GRAVITY (falling) is handled by the PRG by subtracting from z each frame.

Parameters:

ParameterMeaning
heightMaximum step/stair height that is climbed automatically, in DIV2 units. 0 = disabled.

Returns: 0.

Note: Requires collisions to be enabled: does not work if you have called MODE9_COLLISION(0).
Example program: 
MODE9_STEP(70);   // climb steps up to 70 units high

// Gravity is applied by the PRG each frame:
fall_v += 4;            // accelerate the fall
m9.height -= fall_v;    // descend
if (m9.height < 70)     // clamp when hitting the floor
    m9.height = 70; fall_v = 0;
end
See: MODE9_COLLISION() - MODE9_COLLIDE() - MODE9_SET()
