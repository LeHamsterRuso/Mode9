MODE9_SCRW()

MODE9_SCRW()

Description:

Returns the width of the active video resolution in pixels. Useful for adapting the mouse, viewports and the UI to any resolution, not just 320x200.

Can be called at any time (not just at startup), in case you change the resolution with set_mode() during the game.

Returns: The width in pixels of the current screen.

Example program: 
GLOBAL
    scr_x; scr_y;
BEGIN
    set_mode(640480);
    scr_x = MODE9_SCRW();   // 640
    scr_y = MODE9_SCRH();   // 480
    // Centre the crosshair at any resolution
    write(0, scr_x/2, scr_y/2, 4, "+");
END
See: MODE9_SCRH() - MODE9_SPLIT()
