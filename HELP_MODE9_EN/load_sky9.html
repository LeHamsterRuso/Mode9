LOAD_SKY9()

LOAD_SKY9( <MAP texture> )

Description:

Loads a panoramic .MAP texture as the sky. It replaces the flat background colour: the sky rotates with the camera's horizontal rotation (yaw) and shifts with its vertical tilt (pitch), creating a dome-like feeling.

The texture must wrap 360 degrees horizontally (its left and right edges must match up) and uses the same palette as the game. It is deactivated with SKY9_OFF().

Parameters:

ParameterMeaning
MAP texturePanoramic .MAP file (360-degree horizontal wrap).

Returns: 1 if the sky loads successfully, 0 if it fails.

Example program: 
LOAD_WLD9("CASTLE.OBJ", "MAP/CASTLE.MAP", 100);
LOAD_SKY9("MAP/SKY.MAP");   // 360-degree panoramic sky
See: SKY9_OFF() - SET_ENV_COLOR9()
