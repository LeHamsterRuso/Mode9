MODE9_UNITSCALE()

MODE9_UNITSCALE( <units> )

Description:

Decides how large the scene appears. When you model in a 3D application, a wall might measure "2 units"; but in DIV distances are measured on a different scale. This function sets the equivalence: how many DIV units equal one model unit. With 100 (the default), 1 OBJ unit equals 100 game units; with 200, the entire scene is twice as large and you need to walk twice as far to cross it.

Normally you do not need to call this, because you already specify the scale as the third parameter of LOAD_WLD9(). Use it only if you want to change the scale separately.

Parameters:

ParameterMeaning
unitsHow many DIV units equal 1 OBJ model unit. 100 = default; 200 = everything twice as large; 50 = everything half size.

Returns: 0.

Example program: 
MODE9_UNITSCALE(100);   // 1 Blender unit = 100 DIV2 units
See: LOAD_WLD9()
