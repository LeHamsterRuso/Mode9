MODE9_PERSPECTIVE()

MODE9_PERSPECTIVE( <on> )

Description:

Chooses between drawing textures faster or more accurately. On floors and walls seen at a very steep angle, the texture may appear to "wobble" or warp slightly; that happens because the engine uses approximate calculations for speed. Enabling this option performs the calculations exactly and textures appear perfectly straight, but rendering costs more and the game may run a little slower.

It is disabled by default (fast mode) because in most scenes the difference is barely noticeable. Enable it only if that wobble bothers you and your machine has plenty of headroom.

Parameters:

ParameterMeaning
on0 = fast, with a slight wobble on very steep surfaces (default); 1 = perfect textures, somewhat slower.

Returns: 0.

Example program: 
MODE9_PERSPECTIVE(1);   // textures without wobble (slower)
See: MODE9_DOWNSCALE()
