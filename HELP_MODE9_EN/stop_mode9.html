STOP_MODE9()

STOP_MODE9()

Description:

Deactivates the 3D scene render. After calling it, the background layer reverts to DIV2's own: put_screen, put and scroll become visible again normally, making this the right function to call before showing a 2D screen (pause menu, game-over screen, etc.).

The loaded scene, textures and models are preserved: you can re-activate the render with START_MODE9() without reloading anything.

This was the key fix to make put_screen work again: while mode 9 is active, the DLL owns the background layer; when stopped, it restores the engine's 2D background.

Returns: 0.

Note: To change the map you do NOT need to stop mode 9: LOAD_WLD9() hot-reloads the scene without touching the camera or the render state.
Example program: 
// Pause: stop the 3D, show a 2D menu, then resume
if (key(_esc))
    STOP_MODE9();                       // background reverts to 2D
    write(0, 160, 100, 4, "PAUSED - press ENTER");
    repeat
        FRAME;
    until (key(_enter))
    delete_text(all_text);
    START_MODE9(id, OFFSET m9.z, 0);    // resume 3D render
end

While mode 9 is stopped, texts and processes with graphics (sprites) are still visible; what is restored is the engine's 2D background.

See: START_MODE9() - LOAD_WLD9()
