MODE9_OBJ9_READY()

MODE9_OBJ9_READY( <handle> )

Description:

Queries the load state of a model from LOAD_OBJ9() or LOAD_OBJ9_ANIM(), which load in the background.

Useful for showing a "Loading..." bar or waiting until everything is ready before starting the game. Not mandatory: DRAW_OBJ9 already silently ignores models that are not ready.

Parameters:

ParameterMeaning
handleHandle of the model to query.

Returns: 1 = loaded; 0 = still loading; -1 = invalid handle or failure.

Example program: 
h_npc = LOAD_OBJ9("KNIGHT.OBJ", "MAP/KNIGHT.MAP", 100);
// Wait for the model to be ready while showing a message
while (MODE9_OBJ9_READY(h_npc) == 0)
    write(0, 160, 100, 4, "Loading...");
    FRAME;
end
delete_text(all_text);
See: LOAD_OBJ9() - LOAD_OBJ9_ANIM() - DRAW_OBJ9()
