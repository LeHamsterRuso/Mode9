MODE9_REGION()

MODE9_REGION( <n> , <x> , <y> , <width> , <height> )

Description:

Defines the rectangle for region n (1..15), just like define_region() but for mode 9. It is then used as the third parameter of START_MODE9() to draw the scene only in that part of the screen (multiple regions create a split-screen, like mode 8 viewports).

Region 0 is always the full screen. A width or height <=0 removes the region.

Parameters:

ParameterMeaning
nRegion number to define (1..15).
x, yTop-left corner of the region, in pixels.
width, heightSize of the region in pixels (<=0 removes the region).

Returns: 1 if the region is defined, 0 if it fails.

Note: For split-screen with multiple independent cameras, MODE9_SPLIT() is usually more convenient.
Example program: 
// Define a viewport in the top half of the screen
MODE9_REGION(1, 0, 0, 320, 100);
START_MODE9(id, OFFSET m9.z, 1);   // draw 3D only in region 1
See: START_MODE9() - MODE9_SPLIT()
