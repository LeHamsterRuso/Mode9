MODE9_SET()

MODE9_SET( <height> , <pitch> )

Description:

Sets the camera eye height and its vertical tilt (pitch). Equivalent to mode 8's m8.height and m8.angle variables.

You can also write directly to m9.height and m9.angle in the structure: the DLL reads them every frame through the BIND.

Parameters:

ParameterMeaning
heightCamera eye height above the ground, in DIV2 units (equivalent to m9.height).
pitchVertical look angle, in thousandths of a degree. 0 = looking at the horizon; positive = upward; negative = downward (equivalent to m9.angle).

Returns: 0.

Example program: 
// Standing height, looking at the horizon
MODE9_SET(70, 0);

// ...or control pitch with the mouse by writing to the structure:
m9.angle += mouse.y * 100;
if (m9.angle >  45000) m9.angle =  45000; end
if (m9.angle < -45000) m9.angle = -45000; end
See: START_MODE9() - MODE9_BIND() - MODE9_FOV()
