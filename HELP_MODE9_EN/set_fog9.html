SET_FOG9()

SET_FOG9( <near> , <far> )

Description:

Activates fog: distant geometry gradually fades into a colour, just like real mist. It adds a sense of depth and also hides the edge of the scene (instead of objects popping in abruptly, they blend in gradually). It is the same idea as mode 8's set_fog.

The fog COLOUR is the one set with SET_ENV_COLOR9(). Geometry blends with that colour progressively: it looks sharp up to the near distance and is completely tinted (hidden) from far onwards.

Enabling fog automatically activates the far clip at the far value: anything beyond it is not drawn (it is already the background colour).

Parameters:

ParameterMeaning
nearDistance (DIV2 units) up to which there is NO fog (fully sharp).
farDistance (DIV2 units) at which fog is total (everything becomes the background colour).

Returns: 0.

Example program: 
SET_ENV_COLOR9(20, 20, 28);   // fog colour
SET_FOG9(300, 1200);          // sharp up to 300, opaque at 1200
See: MODE9_FOG_OFF() - SET_ENV_COLOR9() - MODE9_FARCLIP()
